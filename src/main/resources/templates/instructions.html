RoundTours - Tours and Travel Landing Page

Round Tours is a free downloadable landing page template basically designed for tour and travel agencies. In this travel template, you'll find components for the main functions related to travel, such as flight search, hotel booking, and holiday package listings. Along with this, we have given an excellent place to showcase the offers and deals of the travel tours business to the customers thus it fits perfectly with your marketing requirements. Round Tour template built on the latest Bootstrap version with minimal use of CSS and JavaScript. This landing page template for today's tour and travel industry will help you build a travel website in no time.

How to Edit HTML Template

You can change this template as per your requirement

The color scheme and shadows of the template are defined in the _var.scss file.

Files for Google Fonts, Bootstrap icons and bootstrap framework have been linked in the _vendors.scss file.

In the main CSS is main.scss, the CSS of all the blocks is written by well commenting. Which you can easily change according to your requirement.

Bootstrap Bundel javascript is in JS folder.
